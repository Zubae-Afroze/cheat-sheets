###Git Cheat Sheet

```sh
git config --global user.name 'name'

git congif --global user.email 'email address'

git log #info about changes and commits

git init #initialize repositories

git status #to check status of the current repo

git add 'filename' #adds the specific file to the staging area

git add 'filename1' 'filename2' 'filename3'

git add . #adds all the changes made to the stagin area

git commit -m 'message here' #commit code in staging with a message

git checkout <commit-hash> #rolls back to supplied commit hash

git branch #lists branches

git branch <branch-name> #creates a new branch

git merge <branch-name> #merges the branch to master

```
